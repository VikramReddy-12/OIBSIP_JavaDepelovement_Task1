# Online Reservation System
This project is a simple **Online Reservation System** used for booking and cancelling train tickets.
It was developed as part of the **AICTE OASIS INFOBYTE Internship (Software Development Domain)**.
##  Features

* Login system for authorized users
* Ticket reservation with travel and passenger details
* Auto-display of train number and train name
* Ticket cancellation using PNR number
* Shows full ticket details before cancellation
* Stores all booking and cancellation data in a database
##  Technologies Used
* HTML, CSS, JavaScript
* PHP / Node.js (backend)
* MySQL Database
* XAMPP / phpMyAdmin
##  How to Run
1. Install **XAMPP** and start **Apache & MySQL**
2. Create a folder named `online_reservation_system` inside:
3.

